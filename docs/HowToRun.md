# Electron
---
## Step 1: Dependencies
- This needs to be done for both root and client folder
1. npm install (From root)
2. cd client
3. npm install (From client folder
4. cd .. (Return to root)
--- 
## Step 2: Run
- npm run
- IndexedDB is used for storage. The app will prompt you for a key for openai. This is stored client side as its the user's key.
